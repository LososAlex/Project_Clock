Проект часы это приложение для пк, в котором реализованы функции таких же приложений для самртфонов:
- Таймер для отсчета времени
- Будильник
- Время, где показывается ваше время и текущая дата и время в других часовых поясах России
- Секундомер
Приложение создано для того, чтобы пользователь мог организовать свой досуг за пк
